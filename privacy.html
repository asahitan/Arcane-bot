1. Introduction

AntiSpam ("the Bot") is committed to protecting your privacy and ensuring the security of the data it processes. 2. Data We Collect

2.1 User Data

  • User IDs: Used to identify and manage user activity on Discord servers.
  • Messages: Messages flagged as spam or violating server rules may be temporarily stored for analysis and action.

2.2 Server Data

  • Server IDs: Used to store server-specific settings and preferences.
  • Configuration Settings: Anti-spam rules and preferences set by server administrators.

2.3 Interaction Data

  • Logs of interactions with the Bot, such as command usage, to improve performance and troubleshoot issues.

3. How We Use Collected Data

The collected data is used to:

  • Detect and mitigate spam or abusive behavior on Discord servers.
  • Improve the functionality and performance of the Bot.
  • Provide troubleshooting and support to users and server administrators.

The Bot does not use collected data for marketing or advertising purposes.

4. Data Sharing

The Bot does not sell, rent, or share your data with third parties. Data may only be shared:

  • When required by law or legal proceedings.
  • With Discord, in compliance with their Terms of Service and Developer Policies.

5. Data Retention

  • Temporary Data: Messages flagged as spam are stored temporarily and deleted within 30days unless required for ongoing investigations.
  • Server Settings: Stored as long as the Bot is active on the server or until removed by the server administrator.
